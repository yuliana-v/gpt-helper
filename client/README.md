# 🧑‍💻 GPT CodeGen UI (React + Chakra UI)

Frontend for the GPT CodeGen system. Provides a secure prompt interface and full history dashboard for your dev team.

---

## 🌐 Features

- 🔐 Firebase authentication (email/password or Google Sign-In)
- 🧠 Prompt generation via backend
- 📄 Prompt type presets (`comment`, `test`, `analysis`)
- 🧾 History search, filters, and detail view
- 🎨 Responsive UI with Chakra UI
- 🗂 State-managed with custom `AuthContext`
